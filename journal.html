It's me again! I have learned so much about coding and computer science so far in CS1! One of the first things I learned was HTML, which is the programming “language” that is used in order to have something in a browser. I also learned about variables early on. A variable is the title/name given to information in your code, and that becomes your variable. Variables are used to run functions. A function is a piece of code that makes something happen. It can also be used multiple times. CSS is a programming language that can be used within HTML. Similarly, Javascript can be put anywhere within an HTML code.While CSS and HTML give styling and structure to websites, Javascript is what you use to code different interactive functions. The terminal is what you use to save your tasks. Github is the code program we are using in class. Lastly, the console is where you can see any mistakes in your code. I am not a master at coding and still have so much to learn! I know what everything does, but I still get confused on how to put things into my code or get different functions to work.
